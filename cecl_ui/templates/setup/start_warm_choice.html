Do you have a WARM workbook for this CU?

The CECL-Migration model can read a prior quarterly CECL-Migration-WARM workbook and pre-fill identity, pools, grades, and historical balances. If this is a brand-new CU and you have no prior WARM file, choose "Start from scratch" to type the identity fields yourself.

Use a WARM workbook

Recommended when prior quarterly WARM files exist. We'll read BS CO DQ Data Enter for charter #, CU name, period ending, state/county, and economic baseline, then drop you on Step 1 with those fields already populated.

Start from scratch

Brand-new CU, or no prior WARM workbook handy. You'll type the identity fields on Step 1 and can still upload a WARM file later on Step 2.
