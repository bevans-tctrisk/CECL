For any quarter-end (Mar/Jun/Sep/Dec) inside the lookback window that has no row in the charge-off history table above, pull the credit union's 5300 call report from the TCT NCUA Solr index and write one row per canonical 5300 charge-off bucket below.
YTD differencing: 5300 charge-offs are reported year-to-date. The amount written at each quarter-end is the quarter-only net charge-off, computed as QTR_YTD − prior QTR YTD within the same calendar year (March = Q1 YTD; June = Q2 YTD − Q1 YTD; September = Q3 YTD − Q2 YTD; December = Q4 YTD − Q3 YTD). Non-quarter month-ends (Jan/Feb/Apr/May/Jul/Aug/Oct/Nov) are left empty — downstream consumers treat absence as zero charge-offs for the month.
