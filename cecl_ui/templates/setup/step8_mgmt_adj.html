Management Adjustments

Optional Q-factor overlays the engine adds on top of the base WARM loss rate. Most CUs leave the per-pool table blank and only return here later when management decides a specific pool needs a manual bump.

Per-pool overlay (optional)

Enter a percentage to add to that pool's base loss rate. Positive values increase the allowance; negative values decrease it. Leave blank or 0 for pools you don't want to adjust. The engine applies the overlay across grades using the standard distribution factor (so the overlay you enter here is the pool-wide weighted average).
